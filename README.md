## Adding a New Tool

1. Create folder: `ep{N}/` (athlete) or `ep{N}p/` (parent)
2. Add `index.html` — use an existing tool as the template
3. `git add ep{N}p/index.html && git commit -m "Add ep{N}p tool" && git push`
4. Live within ~60 seconds at the URL pattern above
5. Update this README table and `sports-resource-library/inventory/master-inventory.json`
